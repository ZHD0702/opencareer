# 情感聊天小助手 (Emotional Chatbot)

一个具备长期记忆能力的情感聊天小助手，基于 LangChain 和 DeepSeek 模型构建。

## 功能特性

- 🎭 **情感支持**：温暖、善解人意的情感陪伴
- 🧠 **长期记忆**：智能记住用户分享的重要信息（个人信息、偏好、重要事件、情感状态、目标）
- 💬 **对话历史**：支持多轮对话，保持上下文连贯
- 📝 **记忆管理**：提供记忆摘要、详细记忆查看和清除功能

## 技术栈

- Python 3.10+
- LangChain (1.2.x)
- DeepSeek API

## 环境要求

### Python 版本
- Python 3.10 或更高版本

```bash

```

## 快速开始

### 1. 配置 API 密钥

在项目根目录创建 `.env` 文件，并添加你的 DeepSeek API 密钥：

```env
DEEPSEEK_API_KEY=your_api_key_here
```

### 2. 运行聊天机器人

```bash
python main.py
```

### 3. 使用命令

运行后，你可以使用以下命令：

- `quit` 或 `exit`: 退出聊天
- `memory`: 查看记忆摘要
- `detail`: 查看详细记忆
- `clear`: 清除所有记忆

## 项目结构

```
.
```

## 注意事项

1. 确保你的网络可以访问 DeepSeek API
2. API 密钥需要通过 DeepSeek 官方平台获取
3. 记忆文件 `chat_memory.json` 会自动创建和更新
4. 如果遇到网络连接问题，请检查网络设置或稍后重试
